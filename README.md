# clformat

A port of the Common Lisp FORMAT function to Python.

# Installation

  $ python setup.py install

# Usage

You can check the version like so:

    >>> import clformat
    >>> clformat.__VERSION__
    0.0.1a1

# Tests

  $ python -c "import clformat; clformat.test()"
